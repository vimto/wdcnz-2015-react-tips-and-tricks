## Why React?
- Simple
- Declarative
- Composable
- Testable
- Fast

Note:
In the end we decided to build Atomic in React because was:
- A simple API with small surface area
- A declarative composable approach which seemed easier to scale
- Testable
- And had good performance out of the box
