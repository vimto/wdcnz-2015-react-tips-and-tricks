## Virtual DOM and Synthetic Events
- A fast internal mock DOM 
- On every update, it:
  + Builds a new DOM substree
  + Diffs it with the old one
  + Computes the miminum set of DOM mutations
  + Batch executes them
- Cross-browser wrapper around the native events
